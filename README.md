# IMDb Sentiment Analysis

A Natural Language Processing (NLP) project to classify movie reviews as positive or negative.

## Results
- **Best Model**: Logistic Regression
- **Accuracy**: ~88%
- **Precision**: ~88% 
- **Recall**: ~88%
- **F1-Score**: ~88%

## NLP Pipeline
1. Text Cleaning (remove HTML, special characters)
2. Tokenization
3. Stopword Removal
4. Lemmatization
5. TF-IDF Vectorization
6. Model Building (Naive Bayes, Logistic Regression)
7. Evaluation & Error Analysis
